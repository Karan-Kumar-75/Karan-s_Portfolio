# Karan Kumar — Portfolio

Personal portfolio website showcasing my projects, skills, and experience in Data Science & Analytics.

🔗 **Live:** [karan-kumar-75.github.io/Karan-s_Portfolio](https://karan-kumar-75.github.io/Karan-s_Portfolio)

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home / Hero with typing animation |
| `about.html` | About me, resume download, stats |
| `projects.html` | Featured projects gallery |
| `contact.html` | Contact cards & social links |
| `404.html` | Custom error page |

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Modular architecture (variables, animations, responsive)
- **Vanilla JS** — Modular scripts (navigation, projects, utils, data-handler)
- **GitHub Pages** — Hosting

## Project Structure

```
├── index.html / about.html / projects.html / contact.html / 404.html
├── assets/
│   ├── css/        → variables.css, style.css, animations.css, responsive.css
│   ├── js/         → main.js, navigation.js, projects.js, utils.js, data-handler.js
│   ├── images/     → profile/, projects/, backgrounds/, icons/, logos/
│   ├── videos/     → hero-bg, project demos
│   ├── documents/  → resume, certificates, reports
│   ├── data/       → excel/, csv/, json/, notebooks/
│   └── fonts/      → custom web fonts
├── projects/       → individual project detail pages
├── downloads/      → public downloadable files
├── components/     → reusable navbar, footer, project-card
├── config/         → site-config.json
└── .github/        → deployment workflows
```
